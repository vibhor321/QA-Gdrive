# Streamlit APP
App to query data from a google drive folder

## Getting Started
1. Clone this repository to your local machine.
2. Install the required dependencies by running pip install -r requirements.txt.
3. Run streamlit run app.py

## Technologies Used
1. Python( >3.8.10 )
2. Langchain
3. Streamlit
4. Generative AI
